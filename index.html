About

My name is André L. S. Bonfante, I'm a software developer with experience mainly on web development. My focus and priorities are always to build high quality software, respecting the best pratices and patterns of software engineering.

Usually, I work with back-end techologies like PHP and C#. But I have experience in front-end as well using technologies like HTML, CSS, Javascript and some SPA frameworks (like Angular).

I have experience with some technologies like:

  • C# (.NET Core, ASP.NET Core - MVC and WEB API; Entity Framework Core)
  • PHP 7+ (Laravel, Lumen e Slim; unit testing with PHPUnit)
  • PostgreSQL, MySQL e MongoDB
  • Git
  • Javascript (Angular 1.x e Angular 8+; jQuery)
  • Typescript
  • HTML
  • CSS

I've worked for some companies like Pro-ip - supporting clients infrastructures and networks, Moderniza - building an ERP, BW Center - building an e-commerce platform, and now I'm currently working to UV Franchising building applications for franchising companies.

In the academic environment, I started my bachelor degree in computer engineering in 2015 at Universidade do Vale do Taquari, which is based on the state of Rio Grande do Sul, Brazil. There I completed 50% of the course grade. Then, in 2021, I decided to change from computer engineering to computer science. Hence, I started my computer science bachelor degree at Universidade Federal da Fronteira Sul, which is based on the state of Santa Catarina, Brazil.
